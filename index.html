Multiple object detection using a pre-trained machine learning model in TensorFlow.js

Hold some objects up close to your webcam or point it at things to get a real-time classification! Up to 80 classes of objects can be detected. At any time when a person is detected, "Humans detected?" will go from "NOPE" to "YES!".

Wait for the model to load, and then when it's ready, you'll be able to click on the Enable Webcam button below - you'll need to give the browser permission to use your camera.
